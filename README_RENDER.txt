SITE DA LARISSA - PRONTO PARA RENDER

Esta versao esta com a FOTO EMBUTIDA dentro do index.html.
Por isso, mesmo que o Render/GitHub tenha problema com caminho de imagem, a foto deve aparecer.

Como publicar no Render:
1. Descompacte este ZIP.
2. Envie para o GitHub os arquivos de dentro da pasta, principalmente index.html, render.yaml e assets/.
3. No Render, escolha New + > Static Site.
4. Configure:
   Build Command: echo ok
   Publish Directory: .
5. Clique em Deploy.

Depois do deploy, abra o link em aba anonima ou atualize com CTRL + F5 para limpar cache.
